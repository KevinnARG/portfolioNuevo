¡Hola! Soy Kevin, un desarrollador FrontEnd nacido en Argentina.

Soy un estudiante autodidacta de 15 años, invertí 2 años en estudiar lo fundamental de desarrollo Front-end. Trabajé en proyectos simples y en alguno un poco más ambicioso y gracias a eso hoy cuento con buena experiencia en el desarrollo.

Gracias a estos proyectos pude conocer gente que le gusta hacer lo mismo que yo y trabajar con ellos, lo que me ayudó a obtener experiencia en trabajo en equipo, flexibilidad, claridad, y colaboración, lo más importante.

Actualmente estoy buscando un nuevo rol como desarrollador.

Tecnologías

Estas son las principales tecnologías que uso en mis proyectos.

Typescript

TypeScript es una extensión al lenguaje de programación JavaScript.

Angular

Angular es un framework para desarrollo web en TypeScript hecho por Google.

SASS

SASS es un pre-procesador del lenguaje de hoja de estilos en cascada.

Javascript

JavaScript es un lenguaje de programación interpretado, es de estándar ECMAScript.

HTML

HTML es un lenguaje de marcado para el desarrollo web, la base de toda página web.

Git

Git es un software de control de versiones. Muy importante en proyectos de gran escala.
